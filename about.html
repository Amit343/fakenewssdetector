About the project

I wanted to test my theory using data science and machine learning. I have compiled Python code that constructs a Passive Aggressive Linear model to predict whether or not an article is fake.The dataset used in this model was created from my dummy data. I am using several features or characteristics about common articles and trying to predict if they are fake or not. In my case, fake is determined by a predetermined label, as well as real.Because both text and numeric features are used in this model, natural language processing (NLP) is beneficial for prediction. I used the common term-frequency inverse document frequency vectorizer to transform the count of text into a weighed matrix of term importance. I then compiled both the TextTransformer and TfidfVectorizer for the text features like title and label.I predicted on my X_test and assigned that to prediction. I used metrics.accuracy_score for determining my accuracy.my accuracy is 88%.
